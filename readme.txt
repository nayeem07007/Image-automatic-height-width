Image Automatic Height Width plugin will fix error "Image Elements do not have explicit width and height"

== Description ==

Image Automatic Height Width plugin will work independently and automatic fix google page inside error "Image Elements do not have explicit width and height"

Very easy to use no other setting for this plugin just install it,  this plugin will find all image which don't have height width,  then set a height width automation 


== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

Step 1: Download and unzip the plugin.
Step 2: Upload the 'image-automatic-height-width.zip' folder into the '/wp-content/plugins/' directory
Step 3: Go to the Plugins admin page and active the plugin.
